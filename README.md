# DRAWING-NAV
DRAWING-NAV: Engineering Drawing Passport standard based on Document Space, Geometry Space, and Semantic Space.
DRAWING-NAV is an open protocol for transforming engineering drawings into structured engineering knowledge.

The protocol separates three layers:

• Document Space — authority, revisions, standards, responsibility.
• Geometry Space — dimensions, tolerances, views, constraints.
• Semantic Space — functions, interfaces, load paths, engineering meaning.
