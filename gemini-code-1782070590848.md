# DRAWING-NAV Manifest

## Mission
To transform static engineering drawings into active, machine-navigable engineering assets.

## Scope
Engineering drawings, assembly schemes, technical specifications, and CAD-derived 2D representations.

## Core Values
1. **Human First:** All artifacts must be readable by engineers without special tools.
2. **AI Compatible:** High semantic density for autonomous agent processing.
3. **Open Interoperability:** Standardized formats (YAML/MD) for CAD/PLM systems.

## Difference from others
- **vs OCR:** We extract meaning, not just text.
- **vs CAD Viewer:** We build a semantic graph, not just pixel/vector display.
- **vs BOOK-NAV:** BOOK-NAV navigates *knowledge*; DRAWING-NAV navigates *objects*.
